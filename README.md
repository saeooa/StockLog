# StockLog

## 프로젝트 소개
**주식 매매 일지 시스템: 스톡로그 **</br>
: 청년층의 주식 투자 참여 증가에 주목하여, 금융과 프로그래밍 개발을 기반으로 주식 매수·매도 기록과 현재 수익률을 관리할 수 있는 일지형 시스템을 개발하는 프로젝트

### 📅 개발 기간
